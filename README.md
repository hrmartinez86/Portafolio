# Portafolio

Este es un portafolio creado para mostrar proyectos y habilidades. Contiene una estructura organizada con diferentes recursos como imágenes, estilos y scripts.

## Estructura del Proyecto

```
index.html
images/
recursos/
  Proyecto web/
    index.html
    css/
      bootstrap.min.css
      estilos.css
    images/
    js/
      bootstrap.min.js
      jquery-3.3.1.min.js
      popper.min.js
    scss/
      estilos.css
      estilos.scss
styles/
  estilo.css
```

## Recursos Utilizados

- **HTML**: Para la estructura del contenido.
- **CSS**: Para los estilos visuales.
- **JavaScript**: Para la interactividad.
- **Bootstrap**: Framework CSS para diseño responsivo.
- **SCSS**: Preprocesador CSS para estilos más organizados.

## Cómo Visualizar el Proyecto

1. Clona este repositorio:
   ```bash
   git clone <URL-del-repositorio>
   ```
2. Abre el archivo `index.html` en tu navegador.
